Простая, но функциональная "Kanban-доска" для управления задачами.
Позволяет:

- Добавлять, редактировать и удалять задачи
- Назначать категории и статусы
- Устанавливать сроки выполнения
- Полностью аутентифицированный доступ.

Основные возможности:

- Регистрация и вход
- Выход из аккаунта
- Создание задачи с названием, описанием, категорией и датой
- Редактирование и удаление задач
- Просмотр задач по статусам («Нужно сделать», «В работе», «Готово» и др.)
- Установка срока исполнения через встроенный календарь.

Аутентификация

- Токен хранится в localStorage
- Проверка токена при загрузке приложения
- Защита маршрутов: если авторизован — редирект на главную.
